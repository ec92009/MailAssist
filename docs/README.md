# MailAssist Viewer

Static GitHub Pages app for viewing committed MailAssist draft and execution-log snapshots in the browser.

The current viewer publishes:

- generated drafts
- execution logs

The current generator writes:

- `site/index.html`
- `site/data.json`

Open `site/index.html` locally for quick testing, or publish the generated `site/` folder through GitHub Pages.

Intended future viewer improvements:

- draft status filters
- revision history
- redacted public snapshots
- thread-level drill-down

Published URL after GitHub Pages is enabled for `ec92009/MailAssist`:

- `https://ec92009.github.io/MailAssist/`
